# Bat

## Themes

- `bat --list-themes`: preview all themes
- Configure using
  - environment variables
    - `BAT_THEME`
    - `BAT_THEME_DARK`
    - `BAT_THEME_LIGHT`
  - configuration file
    - `--theme=<your-theme>`
    - `--theme-dark=<your-dark-theme>`
    - `--theme-light=<your-light-theme>`
