# README

This folder should be a soft link to `~/.config/kitty`.

## Usage: Inline theme switching

From within kitty terminal (!)

```sh
# to dakr them...
./switch-dark.sh

# to light them...
./switch-light.sh
```

## Light themes

Light themes which work ok-ish with programs like `bat`, `ranger`, `colorls`, `exa`, etc.:

- CLRS
- Material
- Tango Light
- Tomorrow
